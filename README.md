# 📊 Student Perfomance Dashboard

## 📌 Project Overview

The Data Analysis Dashboard is a web application developed using Streamlit. It allows users to upload a CSV dataset and perform basic data analysis and visualization through an interactive dashboard.

---

## 🚀 Features

- Upload a CSV dataset
- Preview the uploaded dataset
- View the entire dataset
- Display the top 5 rows
- Display the bottom 5 rows
- Show dataset shape (rows and columns)
- Display column names
- Show statistical summary using `describe()`
- Check missing values
- Check duplicate values
- Display dataset information
- Generate interactive graphs
  - Line Chart
  - Bar Chart
  - Histogram
  - Box Plot

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib

## 📈 Dashboard Workflow

1. Upload a CSV file.
2. Preview the dataset.
3. View dataset shape.
4. Display column names.
5. View statistical summary.
6. Check missing values.
7. Check duplicate values.
8. Display dataset information.
9. Select a graph type.
10. Choose a numeric column.
11. Visualize the selected graph.

---

## 📊 Graphs Supported

- Line Chart
- Bar Chart
- Histogram
- Box Plot
- Area Chart
- Scatter plot
- Pie chart

---

## 🎯 Objective

The objective of this project is to provide a simple and interactive dashboard for exploring, analyzing, and visualizing CSV datasets without writing additional code.
